# Code of Conduct

## Our standard

Sovereign Portal is a working project, not a social club. The bar for participation is simple: **be useful and be civil.**

### Expected

- **Stay technical.** Bugs are bugs, design choices are design choices, opinions are opinions. Critique the work, not the person.
- **Be direct.** Honest, specific feedback is a gift. Hedging and indirection waste everyone's time.
- **Assume good faith.** People who file issues or open PRs are trying to help. Read them generously before pushing back.
- **Own your mistakes.** Code review will find issues with your work. The right answer is "good catch, fixing it," not a defense.
- **Cite the docs.** If you think someone is wrong, point them at the section of the docs that backs you up. If no doc covers it, that's a documentation bug worth filing.

### Not tolerated

- Personal attacks, harassment, or insults of any kind
- Sexualized language or imagery
- Discrimination based on identity (race, gender, sexual orientation, religion, nationality, disability, age, etc.)
- Public or private harassment of contributors or maintainers
- Publishing private information without explicit permission
- Sustained disruption of conversations, issue threads, or community spaces
- Conduct that would reasonably be considered inappropriate in a professional setting

---

## Enforcement

Maintainers will moderate issues, PRs, and discussions to keep them on the standard above. The escalation path:

1. **First instance:** a public reply asking for a course correction, or a private DM if more appropriate
2. **Second instance:** a temporary cooldown — comments hidden, interaction limited, or short-term ban from the issue tracker
3. **Repeated or severe instance:** permanent ban from the project and any associated communities

Maintainers have discretion to skip steps for severe first-time violations (e.g. targeted harassment, threats, doxxing).

---

## Reporting

Report Code of Conduct violations to **support@freshify.io**. Reports are confidential. We will acknowledge within two business days and follow up with the outcome of any review.

If a maintainer is the subject of a report, escalate to **security@freshify.io** — the security alias bypasses normal triage and is read directly by Freshify principals.

---

## Scope

This Code of Conduct applies in all project spaces — GitHub issues, pull requests, discussions, code comments — and in any space where someone is representing the Sovereign Portal project externally.
